# cookie Module

inspired from $.cookie plugin by Klaus Hartl (stilbuero.de)

## get ( name ) → {String | null}

Returns cookie by given name

### Parameters:

* name {String}

### Returns:

{String | null}

## set ( name, val, options ) → {this}

Sets cookie by given name

### Parameters:

* name {String}
* val {String}
* options {Object}

### Returns:

{this}
